## Dataset Playground

stub

The dataset playground lets you open imported datasets (starting with LeRobot
v2.1 / v3 layouts from Hugging Face) in a structured viewer with synchronized
video, low-dimensional plots, and a scrubbable timeline.

### Supported formats

stub

- LeRobot v2.1 / v3 (Hugging Face lazy imports)

### Related API

stub

- `GET /api/v1/datasets/by-slug`
- `GET /api/v1/datasets/{uuid}/episode-manifest`
- `GET /api/v1/datasets/{uuid}/resolve`
