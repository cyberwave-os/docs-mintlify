## What is the Unitree Go2?

The **Unitree Go2** is an intelligent quadruped robot, a bionic dog designed for research, exploration, and real-world applications. It combines advanced AI-driven locomotion with robust mechanical performance, making it one of the most versatile quadrupeds in its class.

**Key specifications:**

| Spec | Detail |
|------|--------|
| **Recognition System** | Ultra-wide 4D LiDAR L1 (360° × 90° FOV) |
| **Max Speed** | ~5 m/s |
| **Peak Joint Torque** | ~45 N·m |
| **Wireless** | Wi-Fi 6, Bluetooth, 4G |
| **Battery** | 8,000 mAh (standard) / 15,000 mAh (extended) |
| **Endurance** | ~2–4 hours depending on configuration |

**Core capabilities:**

- **4D LiDAR perception** — ultra-wide hemispherical field of view with a minimum detection distance of 0.05m for all-terrain recognition and safe navigation
- **AI-powered locomotion** — trained via large-scale simulation to handle complex gaits including obstacle climbing, roll-over recovery, and adaptive walking
- **3D mapping and autonomy** — create point cloud maps and define navigation paths using the built-in LiDAR
- **Intelligent side-follow (ISS 2.0)** — wireless vector positioning with obstacle avoidance for smooth traversal over challenging terrain
- **OTA updates** — cloud-based firmware upgrades that continuously improve performance
