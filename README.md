# Cyberwave Docs

## Overview

This repository contains the Cyberwave documentation site, including quickstart guides, API references, SDK examples, and conceptual overviews. The content is written in Markdown/MDX and organized to mirror the product areas, making it easy to update or extend alongside product changes.

Key directories:
- **quickstart.mdx**: A guided setup to help new users configure and use Cyberwave quickly.
- **api-reference/**: Endpoint reference material with request/response details and usage notes.
- **snippets/** and **sdks/**: Reusable code examples and SDK-specific guidance.
- **concepts/**: Explanations of core Cyberwave ideas to provide context for the rest of the docs.

To contribute, edit the relevant Markdown/MDX files and run your preferred linting or preview workflow before opening a pull request.
