## Overview

The [Unitree Go2](https://cyberwave.com/unitree/go2) is a quadruped robot with 12 joints (3 per leg) and onboard cameras. This example shows how to create a digital twin and control its movement.

<Info>
**Registry ID:** `unitree/go2`

The Go2 has RGB cameras, making it a `CameraTwin` with video streaming capabilities.
</Info>

---

## Quick Start

### Installation

```bash
pip install cyberwave
```

### Create a Twin

```python
import cyberwave as cw

# Create a digital twin of the Go2
# Returns CameraTwin due to camera capabilities
robot = cw.twin("unitree/go2")

print(f"Twin type: {type(robot).__name__}")  # CameraTwin
print(f"Twin UUID: {robot.uuid}")
```

## Movement Control

### Position Control

Move the robot to absolute positions:

```python
import cyberwave as cw

robot = cw.twin("unitree/go2")

# Move to position
robot.move(x=2.0, y=1.0, z=0.0)

# Or use a list
robot.move_to([2.0, 1.0, 0.0])
```

### Rotation Control

Rotate the robot's body:

```python
# Rotate using euler angles (degrees)
robot.rotate(yaw=45)  # Turn 45 degrees

# Rotate to face a direction
robot.rotate(yaw=90, pitch=0, roll=0)
```

## Leg Joint Control

The Go2 has 12 jointsâ€”3 per leg:

| Leg | Joints |
|-----|--------|
| Front Left (FL) | `FL_hip`, `FL_thigh`, `FL_calf` |
| Front Right (FR) | `FR_hip`, `FR_thigh`, `FR_calf` |
| Rear Left (RL) | `RL_hip`, `RL_thigh`, `RL_calf` |
| Rear Right (RR) | `RR_hip`, `RR_thigh`, `RR_calf` |

### Control Individual Joints

```python
import cyberwave as cw

robot = cw.twin("unitree/go2")

# Set front left leg joints
robot.joints.set("FL_hip", 0)
robot.joints.set("FL_thigh", 45)
robot.joints.set("FL_calf", -90)

# Get joint position
thigh_angle = robot.joints.get("FL_thigh")
print(f"FL thigh: {thigh_angle}")
```

### Synchronize All Legs

```python
import cyberwave as cw

robot = cw.twin("unitree/go2")

# Define a standing pose
stand_pose = {
    "FL_hip": 0, "FL_thigh": 45, "FL_calf": -90,
    "FR_hip": 0, "FR_thigh": 45, "FR_calf": -90,
    "RL_hip": 0, "RL_thigh": 45, "RL_calf": -90,
    "RR_hip": 0, "RR_thigh": 45, "RR_calf": -90,
}

# Apply pose
for joint, angle in stand_pose.items():
    robot.joints.set(joint, angle)
```

## Camera Streaming

The Go2's cameras can stream video to the digital twin:

### Start Streaming

```python
import asyncio
import cyberwave as cw

async def stream_camera():
    robot = cw.twin("unitree/go2")
    
    # Start camera stream (requires pip install cyberwave[camera])
    streamer = robot.start_streaming(fps=15, camera_id=0)
    
    await streamer.start()
    print("Streaming started!")
    
    # Stream for 60 seconds
    await asyncio.sleep(60)
    
    await streamer.stop()
    robot.stop_streaming()

asyncio.run(stream_camera())
```

<Note>
Camera streaming requires additional dependencies:
```bash
pip install cyberwave[camera]
```
</Note>

## Real-Time Updates

Subscribe to state changes:

```python
import cyberwave as cw

robot = cw.twin("unitree/go2")

def on_position_update(data):
    print(f"Position: {data}")

def on_joint_update(data):
    print(f"Joints: {data}")

# Subscribe to updates
robot.subscribe_position(on_position_update)
robot.subscribe_joints(on_joint_update)

# Movement triggers callbacks
robot.move(x=1, y=0, z=0)
```

## Walking Simulation

Simulate a simple walking gait:

```python
import time
import cyberwave as cw

robot = cw.twin("unitree/go2")

# Simple alternating leg lift pattern
def walk_step(lift_height=30):
    # Lift front-left and rear-right
    robot.joints.set("FL_thigh", 60)
    robot.joints.set("RR_thigh", 60)
    time.sleep(0.2)
    
    # Lower and lift opposite pair
    robot.joints.set("FL_thigh", 45)
    robot.joints.set("RR_thigh", 45)
    robot.joints.set("FR_thigh", 60)
    robot.joints.set("RL_thigh", 60)
    time.sleep(0.2)
    
    # Return to standing
    robot.joints.set("FR_thigh", 45)
    robot.joints.set("RL_thigh", 45)

# Take 5 steps
for i in range(5):
    print(f"Step {i + 1}")
    walk_step()
    time.sleep(0.3)
```

## Full Example

Complete example with movement and real-time monitoring:

```python
import os
import time
from cyberwave import Cyberwave

def main():
    # Initialize client
    client = Cyberwave(api_key=os.getenv("CYBERWAVE_API_KEY"))
    
    # Create Go2 twin
    robot = client.twin(
        "unitree/go2",
        environment_id=os.getenv("CYBERWAVE_ENVIRONMENT_ID")
    )
    
    print(f"Created twin: {robot.name}")
    print(f"Type: {type(robot).__name__}")
    
    # Set up monitoring
    def on_update(data):
        print(f"Update: {data}")
    
    robot.subscribe(on_update)
    
    # Initial position
    robot.move(x=0, y=0, z=0)
    robot.rotate(yaw=0)
    
    # Move forward
    print("Moving forward...")
    robot.move(x=2, y=0, z=0)
    time.sleep(1)
    
    # Turn right
    print("Turning right...")
    robot.rotate(yaw=-90)
    time.sleep(1)
    
    # Move forward again
    print("Moving forward...")
    robot.move(x=2, y=-2, z=0)
    time.sleep(1)
    
    print("Done!")
    client.disconnect()

if __name__ == "__main__":
    main()
```
