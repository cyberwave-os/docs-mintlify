## Overview

The [SO101](https://cyberwave.com/the-robot-studio/so101) is a 6-degree-of-freedom robot arm from The Robot Studio. This example shows how to create a digital twin and control its joints.

<Info>
**Registry ID:** `the-robot-studio/so101`

The SO101 has 6 joints that can be controlled individually or together for complex movements.
</Info>

---

## Quick Start

### Installation

```bash
pip install cyberwave
```

### Create a Twin

```python
import cyberwave as cw

# Create a digital twin of the SO101
robot = cw.twin("the-robot-studio/so101")

print(f"Twin created: {robot.uuid}")
print(f"Twin name: {robot.name}")
```

## Joint Control

### Set Individual Joints

Control joints by name or index. Positions are in degrees by default:

```python
import cyberwave as cw

robot = cw.twin("the-robot-studio/so101")

# Set joint by name (degrees)
robot.joints.set("shoulder", 45)

# Set joint by index
robot.joints.set("1", 30)
robot.joints.set("2", -15)
robot.joints.set("3", 60)
```

### Using Radians

Pass `degrees=False` to use radians:

```python
import math

# Set joint in radians
robot.joints.set("shoulder", math.pi / 4, degrees=False)
```

### Get Joint Positions

```python
# Get a specific joint position
shoulder_angle = robot.joints.get("shoulder")
print(f"Shoulder: {shoulder_angle} radians")

# Get all joint positions
all_joints = robot.joints.get_all()
for name, position in all_joints.items():
    print(f"{name}: {position}")

# List available joints
joint_names = robot.joints.list()
print(f"Joints: {joint_names}")
```

## Position and Rotation

### Move the Arm Base

```python
# Move to absolute position
robot.move(x=1.0, y=0.5, z=0.0)

# Move to position using a list
robot.move_to([1.0, 0.5, 0.0])
```

### Rotate the Arm Base

```python
# Rotate using euler angles (degrees)
robot.rotate(yaw=90, pitch=0, roll=0)

# Rotate using quaternion
robot.rotate(quaternion=[0, 0, 0.707, 0.707])
```

## Animated Movement

### Sweep Joint Through Range

```python
import time
import cyberwave as cw

robot = cw.twin("the-robot-studio/so101")

# Sweep joint 3 from 0 to 70 degrees
for angle in range(0, 70, 5):
    robot.joints.set("3", angle)
    print(f"Joint 3: {angle}°")
    time.sleep(0.1)
```

### Multi-Joint Animation

```python
import time
import cyberwave as cw

robot = cw.twin("the-robot-studio/so101")

# Define a sequence of poses
poses = [
    {"1": 0, "2": 0, "3": 0},      # Home
    {"1": 30, "2": -15, "3": 45},  # Reach
    {"1": 30, "2": -30, "3": 60},  # Lower
    {"1": 0, "2": 0, "3": 0},      # Home
]

# Execute sequence
for pose in poses:
    for joint, angle in pose.items():
        robot.joints.set(joint, angle)
    time.sleep(0.5)
```

## Real-Time Updates

Subscribe to joint state changes for real-time monitoring:

```python
import cyberwave as cw

robot = cw.twin("the-robot-studio/so101")

def on_joint_update(data):
    print(f"Joint update: {data}")

# Subscribe to joint updates
robot.subscribe_joints(on_joint_update)

# Move a joint - this triggers the callback
robot.joints.set("1", 45)
```

## Full Example

Complete example with initialization, movement, and cleanup:

```python
import os
import time
from cyberwave import Cyberwave

def main():
    # Initialize client
    client = Cyberwave(api_key=os.getenv("CYBERWAVE_API_KEY"))
    
    # Create SO101 twin
    robot = client.twin(
        "the-robot-studio/so101",
        environment_id=os.getenv("CYBERWAVE_ENVIRONMENT_ID")
    )
    
    print(f"Created twin: {robot.name} ({robot.uuid})")
    
    # Home position
    print("Moving to home position...")
    for joint in ["1", "2", "3", "4", "5", "6"]:
        robot.joints.set(joint, 0)
    time.sleep(1)
    
    # Demo movement
    print("Running demo movement...")
    for i in range(0, 70, 5):
        robot.joints.set("3", i)
        print(f"Joint 3: {i}°")
        time.sleep(0.1)
    
    # Return home
    print("Returning to home...")
    robot.joints.set("3", 0)
    
    print("Done!")

if __name__ == "__main__":
    main()
```
