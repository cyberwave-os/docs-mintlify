## Why this exists

Large files can exceed API gateway or application multipart limits and return `413` errors.

## SDK strategy (stub)

- Keep small uploads on direct multipart endpoints.
- Switch large files to signed URL uploads via attachments.
- Finalize the uploaded attachment into the target domain object (for example, an Asset GLB).
